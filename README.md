# CSE321-Operating-Systems
The objectives are - 
- Understanding core concepts of OS structures, process management, and inter-process communication methods.
- Interpreting CPU schedulers and their functions.
- Understanding the concepts and behaviours of threads and examining the concurrency of multithreading.
- Inspecting thread synchronization mechanisms at the OS level.
- Understanding the secondary memory management and file systems concepts and analysing file system implementation.
- Understanding mechanisms of main and virtual memory management of the OS.
- Analysing OS level security threats and the methods used to protect from the threats.
